## Learn Together Docs

A documentation site built with [Next.js](https://nextjs.org) and [Fumadocs](https://fumadocs.dev).

### Getting Started
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.

Check out the docs at [learntogetherdocs.vercel.app/docs](https://learntogetherdocs.vercel.app/docs)
